Biological neurons collect electrical signals at their dendrites, combine them into a summed input, and fire down the axon when the combined signal exceeds a threshold. This implements spatial contrast extraction at the very first processing layer, before the signal ever reaches the brain. PARALLELS TO ARTIFICIAL NEURAL NETWORKS

Artificial neural networks were built explicitly as mirrors of this biological architecture, and the parallels run deep enough to prove something fundamental about the biocompute model.

Each artificial neuron receives multiple weighted inputs. A low weight de-emphasizes a signal. A high weight amplifies it. Layers of neurons connect fully, each node linked to every node in the preceding and subsequent layers. The learning process de-emphasizes connections that carry no useful information. A zero weight severs the link entirely. The network self-prunes as it learns, stripping itself down to the connections that matter.

The activation function, the mathematical rule that determines when a neuron fires, mirrors biological threshold behavior directly. At the midpoint of the input range, output sits at half strength. Below that, the neuron stays quiet. Above it, output climbs toward full strength, following a smooth S-shaped curve. This mirrors exactly how biological neurons moderate their output based on input magnitude. The simplicity of this function is what makes it so powerful computationally. It differentiates cleanly, allowing the system to calculate precisely how much each connection contributed to any given error.

The weight update rule is where the real significance emerges. Error at the output equals the difference between the desired result and the actual result. That error propagates backward through the network in proportion to connection weights. Stronger connections carry more error responsibility because they contributed more to the discrepancy. The updated weight equals the old weight adjusted by the slope of the error at that connection. This is the universal learning algorithm. Biology implements it through synaptic plasticity, the principle that neurons which fire together wire together, strengthening connections that produce useful results and weakening those that produce errors. Silicon implements the identical process through numerical parameter adjustment. The substrate differs. The computation is identical.

This equivalence is the proof point for the entire biocompute thesis. The computational process running in the avatar's nervous system is universal. It operates identically in biological tissue, in silicon registers, in any medium that supports weighted signal propagation and error-driven adjustment. Substrate independence is confirmed.
